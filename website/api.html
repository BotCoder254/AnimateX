Initialization

Create a new instance of VelocityFX:

const vfx = new VelocityFX({
  observeScroll: true,
  defaultDuration: 400,
  defaultEasing: 'ease-out'
});

Configuration

Option Type Default Description
observeScroll boolean true Enable scroll-based animations
defaultDuration number 400 Default animation duration in milliseconds
defaultEasing string 'ease-out' Default easing function
threshold number 0.2 Intersection observer threshold
root Element | null null Intersection observer root element

Methods

animate(element, options)

Animate an element with custom options

vfx.animate(element, {
  animation: 'fade-in',
  duration: 500,
  easing: 'ease-in-out'
});

sequence(elements, options)

Animate multiple elements in sequence

vfx.sequence(elements, {
  delay: 200,
  animation: 'fade-up'
});

observe(element, options)

Observe an element for scroll-based animations

vfx.observe(element, {
  animation: 'fade-in',
  threshold: 0.5
});

Events

Event Description Example
anx:start Fired when animation starts element.addEventListener('anx:start', handler)
anx:end Fired when animation ends element.addEventListener('anx:end', handler)
anx:cancel Fired when animation is cancelled element.addEventListener('anx:cancel', handler)

Utilities

isReducedMotion()

Check if reduced motion is preferred

if (vfx.isReducedMotion()) {
  // Handle reduced motion preference
}

getDefaultOptions()

Get current default options

const options = vfx.getDefaultOptions();

setDefaultOptions(options)

Update default options

vfx.setDefaultOptions({
  duration: 300,
  easing: 'ease-in'
});

interface SequenceOptions {
