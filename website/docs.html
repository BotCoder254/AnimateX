Getting Started

VelocityFX is a lightweight animation framework that allows you to add beautiful animations to your web applications without writing JavaScript code.

Quick Start

npm install velocityfx

Installation

NPM

npm install velocityfx

CDN

<link rel="stylesheet" href="https://cdn.jsdelivr.net/npm/velocityfx/dist/velocityfx.min.css">
<script src="https://cdn.jsdelivr.net/npm/velocityfx/dist/velocityfx.min.js"></script>

Basic Usage

Add animation classes to your HTML elements:

<div class="anx-fade-in">Fade In</div>
<div class="anx-slide-up">Slide Up</div>
<div class="anx-zoom-in">Zoom In</div>

Animations

Entry Animations

  • .anx-fade-in
  • .anx-slide-up
  • .anx-slide-down
  • .anx-zoom-in

Hover Animations

  • .anx-hover-scale
  • .anx-hover-lift
  • .anx-hover-glow
  • .anx-hover-shake
  • .anx-hover-pulse

Triggers

Scroll Triggers

<div data-anx-scroll class="anx-fade-in">
  Appears on scroll
</div>

Click Triggers

<button data-anx-click="shake">
  Click me!
</button>

Responsive Design

Use responsive prefixes to control animations at different breakpoints:

<div class="sm:anx-fade-in md:anx-slide-up lg:anx-zoom-in">
  Responsive animations
</div>

Accessibility

VelocityFX respects user preferences for reduced motion:

/* Automatically handles prefers-reduced-motion */
<div class="anx-accessible">
  Motion-safe animations
</div>

API Reference

JavaScript API

const vfx = new VelocityFX({
  observeScroll: true,
  defaultDuration: 400,
  defaultEasing: 'ease-out'
});
